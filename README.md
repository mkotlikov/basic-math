# Basic Math

A fun and educational single-page application for kids to practice basic arithmetic.

## Features

- **Times Table**: Multiplication practice.
- **Addition up to 25**: Sums perfect for beginners.
- **Subtraction from 25**: Easy subtraction exercises.
- **Multiple Choice**: Every problem has 4 options.
- **Instant Feedback**: Catchy visual cues for correct and incorrect answers.
- **Speed Tracking**: Measures correct answers per minute (APM).
- **Top Speed**: Compete with yourself to beat your personal best!

## How to Play

1. Open `index.html` in any modern web browser.
2. Select an answer from the four choices provided.
3. If correct, the screen will turn green. If incorrect, it will turn red and show the right answer.
4. Press "Next" to move to the next challenge.
5. Watch your "Speed" (correct answers per minute) and try to beat your "Top Speed".

## Technical Details

- **Self-contained**: No external dependencies or framework required.
- **Pure JavaScript**: Built with vanilla JS for speed and compatibility.
- **Persistent Stats**: Uses `localStorage` to save your highest speed.
